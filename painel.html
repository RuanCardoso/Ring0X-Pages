Painel de Chaves de Autenticacao

Criar, ativar e revogar keys de autenticacao com chamadas HTTP diretas.
Padrao do projeto: http://localhost:8443 (dev) e https://ringapi.guaranihub.com (producao). Se sua autenticacao usa cookie de sessao, deixe token em branco. O painel envia requests com credentials: include.

Criar key

Cria uma nova key com duracao em dias (o prazo final eh calculado ao ativar).

Ativar key

Define data de ativacao atual e data final com base na duracao da key.

Revogar key

Desativa imediatamente uma key ativa.

Resultado da operacao

Pronto para executar operacoes.
Painel standalone em HTML + CSS + JS puro.
